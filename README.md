# Sutu (速涂)

Sutu（速涂）是一款轻量化绘图软件，专注于流畅的数字绘画体验。
当前 Release 仅提供 Windows 版本。
使用gpu compute shader笔刷，对显卡/显存有一定要求。
目标是做一个接近ps绘画体验的轻量级项目。
支持ps的abr笔刷文件导入，支持大部分经典笔刷属性。
